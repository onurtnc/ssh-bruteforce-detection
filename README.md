# SSH Brute Force Detection – SOC Mini Project

## Overview
This project demonstrates a basic SOC detection scenario
focused on identifying SSH brute force attacks using log analysis.

## Scenario
A Linux server is targeted by repeated SSH login attempts
from a single IP address within a short time period.

## Detection Logic
- Analyze authentication logs
- Detect repeated failed login attempts
- Trigger a security alert

## Skills Demonstrated
- Log analysis
- Incident detection
- SOC thinking
- MITRE ATT&CK mapping

## Tools & Concepts
- Linux auth.log
- SSH
- Brute Force detection
- SOC alerting
